对话体验

感受不需要被修复

留白不诊断、不建议、不打鸡血。它倾听、映照、记住。每一段对话都承载着之前的记忆，你说过的任何话都不会消散。

"它记得我三天前紧张面试的事，今天主动问我结果怎么样。从来没有一个应用做到过这件事。"

情绪化石

每次倾诉，凝结成一块记忆的切片

对话不再消散。每一次深度倾诉自动沉淀为一块带有温度和纹理的"化石"——在时间轴上汇聚成你独一无二的情绪地貌。

呼吸锚点

先停下来

当情绪激动到打字都打不清楚的时候，
你不需要再说什么。
你需要的是一个帮你「先停下来」的东西。

开始
点击光球，跟着节奏呼吸

4-7-8 呼吸法 -- 吸气 4 秒 -- 屏息 7 秒 -- 呼气 8 秒
这不是冥想课程，只是朋友在旁边轻轻说：慢慢来。

深夜模式

凌晨两点，
整个世界安静下来

深夜是用户最需要陪伴的时刻。不是亮色模式的反转——是一杯热可可色的温暖，保护疲惫的眼睛，同时守住安全感。
