# AI 算命应用 设计色彩报告

> 调研时间：2026-04-14
> 目标用户画像：20-30 岁年轻人，偏娱乐心态，对中国传统文化有亲近感但不狂热
> 产品调性关键词：古风、玄学、神秘、沉浸、趣味、不吓人

## 1. 行业色彩语言

### 1.1 行业色彩惯例

算命/命理类应用的色彩传统遵循中国玄学美学：

- **暗色基调占主导**：深色（墨黑、藏蓝）传递神秘感和权威感，几乎所有算命 App 都采用暗色模式作为主视觉
- **金色/赤金是标配**：金色象征尊贵、神性、占卜的庄重感，是命理类产品的标志性强调色
- **红色点缀**：朱红/朱砂红象征吉祥、驱邪，用于高亮和 CTA
- **传统中国色**：石青、黛绿、藤黄等传统色经常作为辅助色出现
- **用户预期**：打开算命 App 时，用户期待看到"古色古香"的视觉氛围，过于现代简约会丧失氛围感

### 1.2 头部产品用色统计

| 产品 | 主色 | 辅色 | 强调色 | 整体风格 |
|------|------|------|--------|---------|
| 灵机妙算 | 深红 #8B0000 | 暗金 #B8860B | 朱砂红 #FF461F | 传统中式、暗红金 |
| 测测星座 | 紫色 #6A0DAD | 粉紫 #BB86FC | 金色 #FFD700 | 年轻活泼、梦幻星空 |
| 参天 AI | 深蓝黑 #0D1117 | 金色 #D4AF37 | 青绿 #00CED1 | 科技 + 东方融合 |
| FateMaster | 深紫蓝 #1A1A2E | 蓝色 #0F3460 | 金色 #E0A800 | 国际化、暗色系 |

### 1.3 色彩心理学参考

- **目标用户（20-30 岁年轻人）**：偏好有质感但不老气的暗色调，接受新中式美学，喜欢"高级感"而非"土味中国风"
- **文化考量**：金色/红色在中国文化中有强烈的吉祥寓意，用户会将其与"灵验""正统"关联
- **禁忌**：纯黑+白色的极简配色会让人联想到丧葬（文化禁忌）；全绿色容易让人联想到"绿帽子"（需避免大面积使用）；过于鲜艳的荧光色破坏神秘感

## 2. 推荐配色方案（3 套）

### 方案 A："墨金雅韵"（推荐首选）

- **调性关键词**：高级古风 / 低调奢华 / 神秘内敛
- **适合场景**：强调传统文化底蕴、追求"新中式"高级感的算命应用
- **色彩体系**：
  - 主色：#1C1917（墨褐黑）-- 用途：主背景、营造沉浸氛围
  - 辅色：#292524（暖灰褐）-- 用途：卡片背景、次级区域、分区
  - 强调色：#D4A853（赤金）-- 用途：CTA 按钮、重要高亮、品牌色
  - 成功色：#4A7C59（松绿）
  - 警告色：#D4A853（复用金色，与强调色统一语义）
  - 错误色：#C75B5B（暗红）
  - 背景色（亮）：#FBF8F3（米白/宣纸色）
  - 背景色（暗）：#1C1917（墨褐黑）
  - 文字主色：#F5F0E8（暖象牙白）-- 暗色模式
  - 文字次色：#A8A29E（暖灰）-- 暗色模式
- **对比度检查**：金色 #D4A853 在暗底 #1C1917 上对比度约 7.2:1，达到 WCAG AAA；暖白 #F5F0E8 在暗底上对比度约 13.5:1，达到 AAA
- **参考来源**：中国传统色"墨色""赤金"，新中式设计趋势
- **使用建议**：60% 暗底（#1C1917）/ 30% 暖灰区域（#292524）/ 10% 金色点缀（#D4A853）

### 方案 B："朱砂玄夜"

- **调性关键词**：热烈 / 中国红 / 庙宇氛围 / 张力感
- **适合场景**：强调传统庙宇签文感、更浓烈的中国味
- **色彩体系**：
  - 主色：#1A1118（玄墨紫黑）-- 用途：主背景
  - 辅色：#2D1F2D（暗紫褐）-- 用途：次级区域
  - 强调色：#C04040（朱砂红）-- 用途：CTA、签文高亮
  - 成功色：#4A7C59（松绿）
  - 警告色：#D4A853（赤金）
  - 错误色：#C04040（复用朱砂红）
  - 背景色（亮）：#FFF5F0（淡赤白）
  - 背景色（暗）：#1A1118（玄墨紫黑）
  - 文字主色：#F0E6DC（暖杏白）
  - 文字次色：#9E8E8E（暖灰红）
- **对比度检查**：朱砂红 #C04040 在暗底 #1A1118 上对比度约 5.1:1，达到 WCAG AA；暖杏白在暗底上 > 12:1
- **参考来源**：中国传统色"朱砂"、庙宇签筒美学
- **使用建议**：60% 暗底 / 30% 暗紫褐 / 10% 朱砂红强调

### 方案 C："青玄道韵"

- **调性关键词**：道家清雅 / 山水意境 / 仙风飘逸 / 文人气
- **适合场景**：偏文雅淡然路线、强调"道法自然"的清修感
- **色彩体系**：
  - 主色：#141B1F（青黑）-- 用途：主背景
  - 辅色：#1E2A30（深石青）-- 用途：次级区域
  - 强调色：#5B9EA6（石青蓝）-- 用途：CTA、水墨点缀
  - 成功色：#6B8F71（苔绿）
  - 警告色：#C9A96E（淡金）
  - 错误色：#B05050（暗赤）
  - 背景色（亮）：#F5F8F6（淡青白）
  - 背景色（暗）：#141B1F（青黑）
  - 文字主色：#E8EDE9（淡青白）
  - 文字次色：#8A9A92（青灰）
- **对比度检查**：石青蓝 #5B9EA6 在暗底 #141B1F 上对比度约 5.5:1，达到 WCAG AA；淡青白在暗底上 > 12:1
- **参考来源**：中国传统色"石青""黛色"，山水画意境
- **使用建议**：60% 暗底 / 30% 深石青 / 10% 石青蓝点缀

### 方案对比与推荐

| 维度 | 方案 A "墨金雅韵" | 方案 B "朱砂玄夜" | 方案 C "青玄道韵" |
|------|---------|---------|---------|
| 调性 | 高级古风、低调奢华 | 热烈传统、庙宇氛围 | 清雅道风、仙气飘逸 |
| 行业契合度 | 5/5 | 4/5 | 3/5 |
| 差异化程度 | 4/5（新中式少见） | 3/5（红金较常见） | 5/5（清雅路线罕见） |
| 无障碍友好度 | 5/5（金色对比度优秀） | 4/5（红色需注意色盲） | 4/5（青色系对比度略弱） |
| 娱乐感 | 4/5 | 3/5（偏严肃） | 3/5（偏文雅） |

**首选推荐**：方案 A "墨金雅韵"。理由：金色强调色在暗底上对比度最佳（AAA 级），"新中式高级感"与年轻用户的审美最契合，既有传统韵味又不显老气，娱乐感与神秘感平衡最好。

## 3. 字体趋势

### 3.1 行业字体风格

- 算命/命理类 App 普遍使用**衬线字体或书法字体**作为标题，体现传统文化底蕴
- 正文使用清晰的**无衬线字体**保证可读性
- 头部竞品灵机妙算使用楷体/宋体装饰元素；测测使用系统默认字体（偏现代）
- 行业主流：标题用传统/书法风格 + 正文用现代可读字体

### 3.2 推荐字体搭配

| 用途 | 推荐字体 | 备选 | 说明 |
|------|---------|------|------|
| 中文标题/品牌 | LXGW WenKai（霞鹜文楷）| 令东齐伋体 | 开源免费商用，楷体风格有古韵但不过于装饰，Web Font 可用 |
| 中文正文 | Noto Serif SC | Source Han Serif | 衬线体，阅读舒适，有文化气质，Google Fonts 可用 |
| 英文/数字标题 | Fraunces | Newsreader | 有手工质感的衬线体，与古风调性搭配 |
| 英文/数字正文 | Source Serif 4 | Lora | 可读性好，与中文衬线搭配协调 |
| 装饰/签文 | 自定义 SVG 书法 | -- | 少量特殊场景用 SVG 书法字实现 |

## 4. 视觉风格趋势

### 4.1 行业视觉趋势

- **暗色模式主导**：算命/命理类 90%+ 的应用采用暗色主题，营造神秘氛围
- **布局模式**：单列纵向滚动为主（适合手机端），重要内容居中
- **圆角趋势**：中等圆角（8-16px）为主流，过大圆角破坏庄重感
- **动效风格**：缓慢、沉稳的过渡动画（符合"占卜揭示"的仪式感），避免跳跃/弹性动画
- **视觉元素**：水墨渐变、星辰/星空粒子、八卦/太极图形、烟雾/云雾效果
- **装饰手法**：边框纹样（中国传统回纹/云纹）、印章图形、毛笔笔触

### 4.2 风格方向建议

- **推荐风格**：新中式暗色沉浸风（New Chinese Dark Immersive）-- 为什么适合：既有传统文化底蕴，又符合现代审美，年轻用户接受度高
- **暗色/亮色**：主暗色（算命场景天然适合暗色营造氛围感），可选亮色模式作为备用
- **圆角策略**：建议 8-12px -- 依据：中等圆角保持庄重感，不像 20px+ 大圆角那么"可爱"
- **装饰克制**：水墨/云雾效果只在关键节点使用（如结果揭示页），不要满屏装饰

## 5. 风格参考案例（文字情绪板）

### 案例 1：参天 AI（Cantian AI）

- **整体感受**：科技感、冷静、东方底蕴、可信赖
- **配色特点**：深蓝黑底 + 金色点缀，色彩运用克制，强调色稀缺使用
- **排版特点**：大字号标题、宽松行间距、内容区域明确分隔
- **组件风格**：简约卡片、细边框、投影柔和
- **动效特点**：过渡平缓，无花哨动画
- **值得借鉴**：金色强调色的克制使用，暗底的高级感
- **需要规避**：过于现代/科技感，缺乏传统文化视觉元素，"古风味"不足

### 案例 2：灵机妙算

- **整体感受**：传统、丰富、信息密集、功利感
- **配色特点**：暗红/金色为主色调，色彩饱和度高，装饰元素多
- **排版特点**：信息密度高，字号层级多，留白少
- **组件风格**：圆角卡片 + 传统纹样边框，icon 密集
- **动效特点**：常规 App 动效，无特殊设计
- **值得借鉴**：红金配色的传统感，八卦/太极图形元素的运用
- **需要规避**：信息过载、广告充斥、界面拥挤感，让人感觉"不高级"

### 案例 3：新中式茶饮品牌视觉（霸王茶姬/茶颜悦色）

- **整体感受**：清雅、高级、年轻化国风、有温度
- **配色特点**：以米白/暖灰为底，配合单一主色调（如青/赤/金），色彩运用极度克制
- **排版特点**：大面积留白、字体精选、中英文搭配
- **组件风格**：极简卡片、毛笔笔触装饰、印章图章
- **动效特点**：微妙、优雅
- **值得借鉴**：证明了"新中式可以很年轻很高级"，留白 + 传统元素的平衡感
- **需要规避**：茶饮的"清新"感与算命的"神秘"感有冲突，不能直接搬用

### 案例 4：故宫博物院 App / 每日故宫

- **整体感受**：典雅、博物馆级、庄重、文化沉淀
- **配色特点**：暗底（深棕黑/藏蓝）+ 金色/朱红强调，色彩来自实物文物色
- **排版特点**：大留白、竖排文字、传统排版元素
- **组件风格**：极简但有细节（如卷轴式展开、印章水印）
- **动效特点**：缓慢展开、卷轴式过渡、有仪式感
- **值得借鉴**：文物级色彩的运用、卷轴/展开式的交互隐喻、仪式感的动效
- **需要规避**：博物馆级的庄重感过强，对娱乐定位的产品来说可能"太端着"

## 6. 设计红线

- **必须避免**：
  - 大面积荧光色/霓虹色（破坏古风氛围）
  - 过度使用阴影和光效（"游戏风"而非"文化风"）
  - 满屏装饰纹样（视觉噪音、影响可读性）
  - emoji 作为 UI 图标（破坏调性统一）
  - 全绿色方案（文化禁忌）
  - 纯黑 #000 + 纯白 #FFF 的极简搭配（丧葬联想）

- **色彩禁忌**：
  - 绿色不能作为主色或大面积使用（"绿帽子"联想，中国文化禁忌）
  - 纯黑+纯白的极端对比（缺乏温度，与"玄学暖意"不符）
  - 紫蓝霓虹渐变（AI Slop 典型特征，且与古风调性冲突）
  - 过高饱和度的红色（#FF0000 级别的纯红显得廉价，应使用沉稳的朱砂红）

- **风格陷阱**：
  - "游戏古风"陷阱：大量使用金色边框、火焰特效、宝石质感，变成手游而非工具 App
  - "PPT 古风"陷阱：只是在白底上放几个毛笔字和一个八卦图，缺乏沉浸感
  - "过于严肃"陷阱：做成算命大师的严肃咨询工具，忘记了"娱乐轻松"的定位
  - "装饰过度"陷阱：每个元素都加传统纹样/边框/阴影，导致界面沉重不透气
