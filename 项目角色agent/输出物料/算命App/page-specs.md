### 全局共享组件

#### 1. 品牌 Logo 组

- **结构**：太极/八卦 SVG 图标 + "天机" 文字（LXGW WenKai 700）+ "AI Fortune" 英文副标题（Fraunces italic）
- **颜色**：图标和文字使用 `--color-accent-gold`，英文使用 `--color-text-muted`
- **用于**：P01 首页 Hero 区域（大尺寸）、P02 对话页顶栏（小尺寸）、P03 签文水印

#### 2. 金色主按钮（CTA）

- **样式**：`btn--primary`，背景 `--color-accent-gold`，文字 `--color-text-on-gold`
- **状态**：default / hover（亮金+金光阴影）/ active（缩放0.98）/ disabled（灰色）/ loading（文字替换为"卜算中..."旋转图标）
- **最小尺寸**：44x44px 触控区域
- **用于**：P01 "开始算命"、P02 "生成签文"、P03 "分享"

#### 3. 免责声明栏

- **文案**：" 仅供娱乐参考，不构成任何现实决策依据"
- **样式**：`--text-xs`，`--color-text-muted`，居中，底部固定或页面底部
- **用于**：P01 底部、P02 输入框下方、P03 底部

#### 4. 页面过渡动效

- **入场**：fade + translateY(24px) 上移，700ms ease-out-expo，元素依次 stagger 200ms
- **退场**：fade out 300ms
- **页面间跳转**：P01->P02 为向左滑出/滑入；P02->P03 为签文从底部升起（overlay）

### 交互逻辑

1. **选择器联动**：任意选择器变化 -> 调用 `calculateBaZi()` -> 更新八字预览行（fadeIn 300ms）
2. **月份-日期联动**：切换月份时，日期选择器自动调整可选天数（大月30天/小月29天，简化处理统一30天亦可）
3. **"开始算命"点击**：
   - 按钮进入 loading 状态（文字变"卜算中..."，显示旋转图标）
   - 所有选择器 disabled
   - 延迟 500ms 后"跳转"到 P02（demo 中用状态切换模拟，实际可用 `window.location`）
4. **prefers-reduced-motion**：关闭所有动画，元素直接显示

"此命木旺水相，如春日之林，生机勃勃却需防风折。今年运势整体平稳向上，宜积蓄力量、广结善缘。"

总体运势（详细）：
"命主日元甲木，坐子水长生之地，得月令寅木之助，又有年支亥水相生，可谓根深叶茂。然八字木旺水多而缺金缺土，如林木虽盛却无斧斤修剪、无沃土固根。今年流年运势平稳中带上升之机，上半年宜蛰伏蓄力，下半年贵人运渐起，把握良机可有所突破。建议多接触五行属金、属土之人或事物，以补命中不足。"

性格分析（摘要）：
"木旺之人，仁厚正直，心地善良，如青松般坚韧不拔。思维敏捷，创造力强，但有时过于固执。"

性格分析（详细）：
"甲木日主之人，性格如参天大树，正直向上、光明磊落。为人仁慈宽厚，乐于助人，天生具有领导气质。水多木浮，有时想法过多难以聚焦，宜修心养性。木旺缺金，性格中少了几分果断决绝，在抉择之际容易犹豫。建议日常可多做需要决断力的小事来锻炼魄力。社交方面广结善缘，但要警惕'烂好人'倾向，学会适时拒绝方为上策。"

事业运（摘要）：
"事业如春笋破土，上升势头明显。宜从事文教、创意、科技类行业，贵人在西北方。"

事业运（详细）：
"命主事业宫得寅木当令，印星有力，说明学习能力极强，适合知识密集型行业。木之特性主生发、创造，从事文化创意、教育培训、科技研发、内容创作等行业可事半功倍。今年下半年事业运势渐旺，可能迎来一个重要的职场机遇或项目合作。贵人方位在西北（乾位属金，恰好补命中金之不足）。切忌急功近利，春种秋收，耐心等待是今年事业发展的关键词。"

感情运（摘要）：
"感情如春风拂柳，温柔多情。已有伴侣者感情稳中有升，单身者桃花在秋季。"

感情运（详细）：
"命主八字水木相生，感情细腻丰富，对待感情认真负责。甲木日主的人在感情中往往付出较多，有时过于理想化。今年桃花运主要集中在农历八九月（酉金月），届时社交活动中有望遇到心仪之人。已有伴侣者今年感情和谐，适合共同规划未来大事（如旅行、置业等）。需注意的是，木旺之人有时太过坚持己见，在感情中宜多倾听对方想法，'水善利万物而不争'方为相处之道。"

财运（摘要）：
"正财稳健有余，偏财不宜强求。理财以稳为主，忌冲动投资，下半年有进财之机。"

财运（详细）：
"八字中财星（土）为用神但力弱，说明正财可得但需努力争取。偏财运势平平，不宜参与高风险投资或投机。今年上半年财运一般，以维持现状、积累为主。下半年财运逐渐好转，尤其农历七八月间有一笔意外之财入账的可能（可能来自副业收入、奖金或合作分成）。理财建议：稳健为主，可适当配置固定收益类产品。'君子爱财，取之有道'，保持正当渠道收入最为稳妥。"

### 交互逻辑

1. **页面加载**：
   - 读取 P01 传来的八字数据，渲染排盘区
   - 触发 AI 首轮分析（模拟 1.5s 延迟后显示）
   - 打字机效果逐字显示开场白（30ms/字）-> 五维卡片 stagger 入场

2. **卡片展开/折叠**：
   - 点击卡片任意区域 -> 切换展开/折叠状态
   - 展开动画：max-height 从 0 过渡到内容高度，300ms ease-out-quart
   - 同一时间只展开一个卡片（手风琴模式）

3. **发送消息**：
   - 输入框非空时，回车或点击发送按钮
   - 用户气泡 fadeIn -> 对话区 scrollToBottom -> 显示"卜算中..."
   - 模拟 1-2s 延迟 -> AI 气泡打字机效果显示
   - 轮次计数 +1

4. **快捷追问**：
   - 点击标签 -> 内容填入输入框 -> 自动触发发送逻辑
   - 已使用的标签变灰色（disabled 样式）

5. **重试机制**（AI 回复失败）：
   - 显示重试气泡，包含"大师被打断了"文案和"重试"按钮
   - 点击重试 -> 重新触发 AI 响应
   - 最多 2 次，第 3 次显示"天机不可再泄，请稍后再试"

6. **生成签文**：
   - 点击"生成签文" -> 打开 P03（overlay 从底部升起）
   - 传递当前运势摘要数据

7. **重新算命**：
   - 点击"重新算命" -> 确认弹窗"确定要重新算命吗？当前对话将清除" -> 确认后跳转 P01

8. **八字排盘折叠**：
   - 点击折叠按钮 -> 排盘区收起为单行摘要，300ms 过渡动画
   - 再次点击展开

9. **移动端键盘适配**：
   - 输入框获取焦点时，底部操作区随键盘上推（使用 visualViewport API 或 CSS env(safe-area-inset-bottom)）
   - 键盘弹出时对话区自动滚到底部
   - 输入框失焦时恢复原位

10. **prefers-reduced-motion**：关闭打字机效果和 stagger 动画，内容直接显示
