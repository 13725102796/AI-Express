05 / Motion

动效基调

克制的微交互。使用 exponential easing（ease-out-quart），不使用 bounce/elastic。 Dashboard 场景优先数据呈现速度，动效为辅。Hover 查看效果。

entrance / 600ms
入场动画：translateY(16px) + fade
ease-out-expo
hover / 200ms
悬浮上移：translateY(-4px) + shadow
ease-out-quart
active / 100ms
按下缩放：scale(0.98)
