页面底部信息区

评分说明

综合评分 = 稳定性得分（连续分红年数归一化，权重 40%）+ 股息率得分（近 3 年平均股息率归一化，权重 35%）+ 分红规模得分（累计分红总额归一化，权重 25%）。总分范围 0-100 分。归一化方法：(当前值 - 最小值) / (最大值 - 最小值) * 100。

数据来源

公开市场分红数据（AKShare）

本站数据仅供参考，不构成投资建议。投资有风险，入市需谨慎。

06

响应式与无障碍

Mobile-First 设计，3 个断点覆盖手机、平板、桌面。触控设备自动适配。

断点系统

断点 范围 媒体查询 主要变化
手机 375px - 640px 基础样式 单列布局、表格横向滚动、堆叠式统计卡
平板 641px - 1024px min-width: 640px 2 列网格、表格完整展示、统计卡并排
桌面 1025px+ min-width: 1025px 3 列色板、最大宽度 1200px 居中

触控目标最小尺寸

44px
所有可点击元素至少 44x44px。按钮、Tab、输入框在触控设备上自动增大到 48px。

无障碍支持

:focus-visible 键盘焦点环，不影响鼠标用户
prefers-reduced-motion 减弱动画偏好
所有文字对比度 WCAG AA（4.5:1 正文，3:1 大文字/UI 组件）
表格 aria-label 和语义化表头
@media (pointer: coarse) 触控设备适配
