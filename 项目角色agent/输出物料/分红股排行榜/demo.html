动效基调

克制、流畅、有目的的微交互。所有动画支持 prefers-reduced-motion 降级。

入场动画 (Fade In Up)

500ms / ease-out / translateY(16px)

Hover 上浮效果

200ms / ease-out / translateY(-6px)

骨架屏加载

1.5s shimmer / infinite loop

动效时间令牌

--duration-fast
150ms -- hover/focus 状态变化
--duration-normal
200ms -- Tab 切换、卡片上浮
--duration-slow
300ms -- 入场动画、页面过渡
--easing-out
cubic-bezier(0, 0, 0.2, 1) -- 所有出场/结束动画的默认缓动
