## 1. 全局共享组件规格

### 1.1 顶部导航栏（.nav）

- **功能**：品牌标识 + 可选返回按钮 + 可选副标题/状态文字
- **元素**：
  - 左侧：返回按钮（带页面历史时显示，文案"← 降维返回"或"← 返回主枢"）OR 品牌「紫微灵犀 / [副标]」
  - 右侧：可选副标题或状态徽章
- **样式**：sticky 顶部，高度 56px，毛玻璃 backdrop-filter: blur(10px)，底边金色细线
- **状态**：默认 / 滚动后（背景从透明变为半透明黑）

### 1.2 底部 Tab 栏（.tab-bar）

- **功能**：4 Tab 主导航
- **元素**：
  - 主页（⌂）→ P01-home.html
  - 排盘（✧）→ P02-paipan.html
  - 解盘（◎）→ P03-reading.html
  - 本我（👤）→ P04-profile.html
- **样式**：fixed 底部，高度 64px，金色细线分隔，激活状态金色高亮
- **状态**：当前页对应 Tab 高亮（color: var(--c-gold)），其余 30% 不透明白色

### 1.3 星空背景层

- `.starfield` SVG 星空小背景（fixed inset:0, z-index:-2）
- `.bg-ambient` 顶部径向金色渐变光晕（fixed inset:0, z-index:-1）
- 全屏页（如 P02 命盘、P05 登录）可叠加 Canvas 动态星空

### 1.4 积分图标 / 数字（统一规范）

- 数字字体：Noto Serif SC weight 300，大号字（32-44px），color: var(--c-gold)
- 单位符号：`¤`（古钱币符号），同 var(--c-gold)
- 数字千分位逗号格式化（如 `2,850 ¤`）
- 数字变化时执行 scale(1.15) → scale(1) 的脉冲动画 + textShadow 金色光晕
- 禁止使用 demo.html 的 `.coin` 渐变圆形图标

### 1.5 主 CTA 按钮（.action-btn）

- 金色 1px 描边 + 透明背景 + 内阴影金色微光
- letter-spacing: 4-6px，字号 14-16px
- hover/active：背景变 rgba(212,175,55,0.1) + 外发光 box-shadow
- 圆角 99px（药丸形），padding: 16-18px

### 1.6 AIGC 标识水印组件（PRD Q7 三层防护硬约束）

- **位置 1（页面级）**：所有 AI 解读相关页面右下角固定徽章「AI · gemini-2.5-pro」（不可关闭）
- **位置 2（卡片级）**：解读内容卡片底部固定声明栏（已在 P03 实现）
- **位置 3（分享级）**：分享卡片必须包含水印「本内容由 AI 生成 · 仅供文化娱乐参考」（不可裁剪，与卡片一体渲染）

**DOM 模板（位置 1）**：

```html
<div class="aigc-badge-fixed">
  <span class="aigc-icon">⚠</span>
  <span class="aigc-text">AI · gemini-2.5-pro</span>

<style>
```

## 4. 管理后台页面规格（5 页，遵循同一黑金风格但布局向 dashboard 靠拢）

### 通用布局

- 左侧侧边栏（160px 宽）：金色 logo + 5 个菜单项（模板管理 / 用户管理 / 积分配置 / 数据概览 / 退出）
- 顶部状态栏（48px）：当前管理员名 + 退出按钮
- 主内容区：max-width 1200px
- 配色：仍用 `--c-gold` + `--c-bg`，但 form-bg 适度提亮表格可读性
- 字体：Noto Serif SC（标题）+ 数字用等宽（可选 Cinzel）

### A01 — 管理员登录

- **核心功能**：管理员独立登录入口
- **元素清单**：
  - 中央居中表单（同 P05 极简风但去掉用户端文案"灵犀密脉"）
  - 标题"紫微灵犀 · 主控核心"
  - 用户名 + 密码输入
  - "进入主控核心" CTA
  - 警示文字"未授权访问将被记录"
- **状态**：空 / 输入中 / 加载 / 错误（账号或密码错误）/ 成功
- **交互**：调 `/api/v1/admin/auth/login` → 成功跳 A05 数据概览（默认首页）
- **假数据**：admin / **********
- **跳转**：→ A05

### A02 — 模板管理 CRUD

- **核心功能**：模板列表 + 新建 + 编辑 + 上下架 + 软删
- **元素清单**：
  - 页面标题"神谕模块管理"
  - 顶部操作条："+ 新建模块" 按钮 + 状态筛选 + 搜索框
  - 表格列：ID / 名称 / 标签 / 所需积分 / 状态（上架/下架）/ 已启封人数 / 创建时间 / 操作（编辑/上下架切换/删除）
  - 新建/编辑 modal（半屏 drawer）：
    - 名称（input）
    - 简介（textarea ≤ 200 字）
    - 详细说明（textarea ≤ 1000 字）
    - 提示词正文（textarea ≤ 2000 字，支持占位符 {{排盘数据}} / {{用户性别}} / {{用户出生信息}}）
    - 标签（多选 chips）
    - 所需积分（number）
    - 预览图 URL（input）
    - 排序权重（number）
    - 状态默认下架
- **状态**：空 / 加载 / 正常列表 / 表单错误（名称重复）/ 删除确认
- **交互**：
  - 表格行操作 → CRUD API
  - 状态切换：直接调 PATCH `/admin/templates/{id}/status`
  - 删除：弹确认 → 软删
- **假数据**：7 张表格行（PRD Q3 预设模板）
- **跳转**：← 侧边栏 / → 各 CRUD API

### A03 — 用户管理（查看，P1）

- **元素**：用户列表表格（ID / 手机号脱敏 / 注册时间 / 积分余额 / 已解锁数 / 报告数 / 操作"查看"）+ 搜索 + 分页
- **状态**：空 / 加载 / 正常 / 错误
- **交互**：纯查看，不可修改
- **假数据**：50 条假用户记录
- **跳转**：← 侧边栏

### A04 — 积分配置

- **核心功能**：积分各项配置项的查看和修改
- **元素清单**：
  - 标题"灵犀经济配置"
  - 配置卡片网格（每个配置项一个卡片）：
    - register_bonus / checkin_day_1~7 / share_reward / share_daily_limit / ad_reward / ad_daily_limit / invite_reward / reading_cost
    - 每个卡片：key + 当前值 + 描述 + "编辑"
  - 编辑 modal：直接修改 value（int）
  - 修改后即时生效提示"配置已生效"
- **状态**：空 / 加载 / 正常 / 编辑中 / 错误
- **假数据**：14 项 PRD 预设配置
- **跳转**：← 侧边栏

### A05 — 数据概览

- **核心功能**：数据卡片 + Top 5 模板列表（PRD Q6）
- **元素清单**：
  - 6 张数字卡片网格：
    - 累计注册缘主数
    - 今日新增缘主数
    - 累计神谕推演次数
    - 今日推演次数
    - 累计灵犀发放
    - 累计灵犀消耗
  - 模板启封 Top 5 列表
  - **不做**复杂图表（PRD Q6 决策）
- **状态**：空 / 加载 / 正常 / 错误
- **假数据**：12,345 / 56 / 23,456 / 234 / 1,234,500 / 234,500 + Top 5 模板
- **跳转**：← 侧边栏（默认首页）
