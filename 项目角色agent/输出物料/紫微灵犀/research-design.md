## 1. 行业色彩语言

### 1.1 行业色彩惯例

中国玄学 / 命理 / 国风类产品在色彩上有相对稳定的语义传统：

| 色系 | 在玄学语境的含义 | 常见用法 |
|------|----------------|---------|
| **深紫色** | 紫气东来 / 帝王 / 高贵神秘 | 紫微斗数本身的"紫"字契合，主色首选 |
| **金色（暖金/古金）** | 财富 / 智慧 / 永恒 / 古典权威 | 强调色、积分元素、CTA |
| **暖白 / 米色 / 宣纸色** | 留白 / 雅致 / 古书韵味 | 背景色，承载内容 |
| **墨色（深近黑）** | 墨韵 / 沉稳 / 古典书法 | 文字主色 |
| **朱砂红 / 中国红** | 喜庆 / 警示 / 重要标记 | 慎用，只作点缀（避免与"凶卦"误读） |
| **青色 / 黛色** | 清雅 / 古朴 | 辅助色或边框 |

**禁忌**：
- **避免大面积纯黑**（容易触发"丧"、"晦气"心理联想）
- **避免饱和度过高的纯红配色**（在命理语境会被读作警告/凶卦）
- **避免霓虹绿/荧光色**（与东方雅致调性冲突）

### 1.2 头部产品用色统计

| 产品 | 主色 | 辅色 | 强调色 | 整体风格 |
|------|------|------|--------|---------|
| 文墨天机 | 米白 / 浅金 | 黑 / 灰 | 红（标星） | 工具感强、传统排盘界面、视觉密度极高 |
| 神机阁 | 暖白 / 浅金 | 黑（文字） | 朱红（重点星） | Web 工具感、传统中式 |
| 测测（紫微/星座） | **深紫 / 蓝紫** | 银白 / 浅紫 | 金（会员/高级） | 神秘感 + 年轻化、卡片化 |
| 准了 | 米色 / 暖白 | 灰棕 | 橙黄 | 心理学专业感、留白克制 |
| FateTell（海外） | 深紫 + 黑 | 金 + 银 | 紫色渐变 | 神秘 / 报告型 / 海外科技感 |

**关键洞察**：
- **深紫 + 金**是"AI 命理"赛道事实上的黄金组合（测测、FateTell 已验证）
- 文墨天机/神机阁的"米白 + 朱红"是**传统工具派**用色，但视觉陈旧
- 紫微灵犀 PRD 9.1 已选择"深紫 #2D1B69 + 金 #D4A84B"，**契合赛道认知且能差异化于工具派**

### 1.3 色彩心理学参考

- **目标用户偏好**：22-35 岁，对"高级感 + 神秘感 + 不土"敏感
- **紫色心理学**：紫色是温暖红 + 冷静蓝的混合，**既神秘又克制**，适合"理性思考非理性话题"的紫微解读场景（设计达人）
- **金色心理学**：传达**价值感 + 稀缺感**，与积分系统的"虚拟货币"语义高度契合
- **文化禁忌**：
  - **白色**在传统中式文化有"丧"的联想，但暖白 / 米白 / 宣纸色是**雅致**，不是丧白，需区分色温
  - **黑色**单独大面积会显丧，必须搭配金或紫平衡
  - 避免"绿配紫"（中国传统色彩配伍中视为不雅）

## 2. 推荐配色方案（3 套）

### 方案 A：紫金御阁（PRD 已选基础上的增强版，**首选推荐**）

- **调性关键词**：东方雅致 / 古典智慧 / 高级神秘 / 温暖亲近
- **适合场景**：紫微命理、AI 国学解读、高端文化产品
- **色彩体系**：
  - 主色：**#2D1B69**（深紫 / 紫微御阁）— 品牌色 / 顶部导航 / 主视觉块
  - 主色浅调：**#5B4A99**（紫微亮调）— 次要按钮 / hover
  - 辅色：**#8B7BC9**（雾紫）— 分区背景 / 卡片浅底 / 标签
  - 强调色：**#D4A84B**（古金 / 紫微金）— CTA 按钮 / 积分图标 / 重要数字
  - 强调色亮调：**#E8C871**（淡金）— 高亮文字 / 链接 hover
  - 成功色：**#5B8C5A**（青苔绿）— 签到成功 / 积分获得提示
  - 警告色：**#C9913E**（琥珀）— 积分不足提示
  - 错误色：**#A93838**（朱砂红）— 表单错误（避免大面积使用）
  - 背景色（亮）：**#FFF8F0**（暖宣纸白）— 主页面背景
  - 背景色（暗）：**#1A1025**（紫黑）— 深色模式 / 命盘背景
  - 表面色（卡片）：**#FFFFFF**（亮模式）/ **#2A1F3D**（暗模式）
  - 文字主色：**#1F1233**（紫黑墨）— 标题、正文（亮模式）
  - 文字次色：**#6B5B85**（紫灰）— 副文本、说明
  - 文字反色：**#F5EBD9**（暖白）— 暗模式文字
  - 边框/分割：**#E8DFD0**（亮模式米色边）/ **#3D2E5C**（暗模式紫边）
- **对比度检查**：
  - 主色 #2D1B69 在 #FFF8F0 上对比度 **13.2:1**（WCAG AAA 通过）
  - 强调色 #D4A84B 在 #FFF8F0 上对比度 **2.8:1**（仅适合大字号 / 装饰，不适合正文）
  - 文字主色 #1F1233 在 #FFF8F0 上对比度 **17.1:1**（AAA 通过）
- **参考来源**：测测 App、FateTell、PRD 9.1 既定基调
- **使用建议**（60-30-10 法则）：
  - 60% 暖宣纸白（#FFF8F0）背景 / 暗模式紫黑（#1A1025）
  - 30% 深紫（#2D1B69）顶部导航 + 卡片标题区 + 命盘外圈
  - 10% 古金（#D4A84B）CTA + 积分图标 + 重要数字 + 装饰线

### 方案 B：墨韵青莲（更"国风画卷"导向）

- **调性关键词**：水墨 / 雅集 / 文人画 / 极致克制
- **适合场景**：偏文化 / 学者向产品（紫微灵犀偏年轻向，**不作首选**）
- **色彩体系**：
  - 主色：**#3D3654**（黛紫墨）
  - 辅色：**#7A8B96**（青烟灰）
  - 强调色：**#B8956A**（古铜金）
  - 背景色（亮）：**#F8F4EA**（陈纸黄）
  - 背景色（暗）：**#1F1B2E**
  - 文字主色：**#2A2438**
- **对比度**：主色对背景 **9.8:1**，强调色 **3.2:1**
- **使用建议**：偏古典文人画风，缺乏年轻活力，**不适合 22-35 岁主目标人群的兴奋感**

### 方案 C：星河紫宇（更"科技未来"导向）

- **调性关键词**：星空 / 银河 / 神秘宇宙 / 现代科幻
- **适合场景**：纯星座类产品 / 海外用户
- **色彩体系**：
  - 主色：**#1A0B3D**（深空紫黑）
  - 辅色：**#6E5BCC**（电紫）
  - 强调色：**#FFD700**（亮金）/ **#A8E6FF**（星银蓝）
  - 背景色（亮）：**#F2EEFA**（淡紫白）
  - 背景色（暗）：**#0B0420**（近黑紫）
  - 文字主色：**#0F0824**
- **对比度**：主色对暗背景对比度低，仅适合纯暗色模式
- **使用建议**：**不推荐**，因为偏向"星座 / 占星"语境，与"紫微国学"调性偏离

### 2.1 方案对比与推荐

| 维度 | 方案 A：紫金御阁 | 方案 B：墨韵青莲 | 方案 C：星河紫宇 |
|------|-----------------|-----------------|-----------------|
| 调性 | 东方雅致 + 现代极简 | 文人画 + 极致克制 | 星空科幻 |
| 行业契合度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐ |
| 与 PRD 9.1 一致性 | ⭐⭐⭐⭐⭐（直接对齐） | ⭐⭐ | ⭐⭐ |
| 差异化程度（vs 文墨天机） | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 无障碍友好度 | ⭐⭐⭐⭐⭐（主色 13:1） | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| 22-35 岁用户接受度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |

**首选推荐**：**方案 A 紫金御阁**

**理由**：
1. **与 PRD 9.1 完全对齐**（深紫 #2D1B69 + 金 #D4A84B 是 PRD 既定基调，调研只是补全完整色板）
2. **行业认知契合**（测测 + FateTell 已验证"深紫+金"是 AI 命理黄金组合）
3. **差异化清晰**（明显区别于文墨天机的米白工具派）
4. **无障碍可达**（主色对比度 13:1 远超 AAA 标准）
5. **支持暗色模式**（紫黑色 #1A1025 已规划好，对应 PRD 9.1）

## 3. 字体趋势

### 3.1 行业字体风格

- **传统命理类产品**：常用宋体（强调古典）+ 黑体（保证可读性）
- **AI 解读类产品**：字体层级明确，标题使用书法韵味字体，正文必须无衬线（屏幕阅读优化）
- **2025 趋势**：思源系列字体在国内中文产品占有率高（开源 + 免费商用 + 7 字重）

### 3.2 推荐字体搭配

| 用途 | 推荐字体 | 备选 | 说明 |
|------|---------|------|------|
| 中文标题（带书法韵味） | **思源宋体 Heavy / Bold**（Source Han Serif） | 字体圈欣意吉祥宋（更圆润古风）/ 黄令东齐伋体（康熙字典体感） | 屏幕显示优化的宋体，免费商用，7 字重，开源 |
| 中文正文 | **思源黑体 Regular / Medium**（Source Han Sans） | 苹方（系统） / OPPOSans（免费商用） | 现代无衬线，可读性强 |
| 中文小字 / 标签 | 思源黑体 Light | PingFang SC | 14px 以下使用 Light 字重避免压迫感 |
| 英文/数字标题 | **Playfair Display**（衬线，文学感）或 **EB Garamond**（古典） | Cinzel（罗马碑刻感） | 与中文宋体呼应，营造古典权威 |
| 英文/数字正文 | **Inter** 或 **Source Sans Pro** | system-ui | 与思源黑体呼应，屏幕阅读最佳 |
| 数字（积分 / 命盘大数字） | **DIN Pro** 或 **Manrope** | Inter | 等宽数字 + 现代感，强调"价值" |
| 装饰用 | 装甲明朝体（标题装饰偶尔使用） | / | 硬朗霸气，适合 hero 区点睛 |

**字号层级建议**（移动端 375px 基准）：

| 层级 | 字号 | 字重 | 行距 |
|------|------|------|------|
| H1 主标题 | 28px | 思源宋体 Bold | 1.4 |
| H2 区块标题 | 22px | 思源宋体 SemiBold | 1.4 |
| H3 卡片标题 | 18px | 思源黑体 Medium | 1.5 |
| 正文 | 15px | 思源黑体 Regular | 1.7 |
| 副文本 | 13px | 思源黑体 Regular | 1.6 |
| 标签/说明 | 12px | 思源黑体 Light | 1.5 |
| 命盘宫位文字 | 11-13px | 思源黑体 Medium | 1.4 |
| 积分大数字 | 32-40px | DIN Pro Bold | 1.0 |

## 4. 视觉风格趋势

### 4.1 行业视觉趋势（2025）

- **布局**：移动端首选**底部 Tab + 全屏卡片流**；命盘类产品多用**居中宫位图 + 滚动详情**
- **风格**：扁平化为主，但加入**微弱的渐变 + 阴影**营造层次（避免纯扁平的廉价感）
- **暗色 / 亮色**：命理类产品**暗色模式占比上升**（与"夜间使用 + 神秘感"场景契合），但默认推荐亮色（暖宣纸白）以降低初次使用门槛
- **圆角**：中等圆角（**12-16px**）是主流，避免极端大圆角（过于活泼）或直角（过于工具）
- **纹样**：**克制使用**中式纹样，如祥云 / 卷草 / 如意纹，仅作点缀（边框装饰、卡片角标），不做满铺背景
- **动效**：流式输出 + 入场淡入 + hover 缩放，避免过度动画

### 4.2 风格方向建议

| 维度 | 建议 | 依据 |
|------|------|------|
| 推荐风格 | **东方现代极简（Modern Oriental Minimalism）** | 与 PRD 9.1 一致；区别于文墨天机的"工具感"和测测的"年轻化神秘感" |
| 默认主题 | **亮色（暖宣纸白）** | 降低初次使用门槛，PRD 9.1 已定义 |
| 暗色支持 | **必须支持**（紫黑 #1A1025） | 命理用户夜间使用频次高，PRD 9.1 已规划 |
| 圆角策略 | **卡片 16px / 按钮 12px / 输入框 10px / 命盘宫位 8px** | 营造现代感的同时保留克制 |
| 阴影策略 | **轻阴影 + 深紫色调阴影**（rgba(45,27,105,0.08)） | 避免黑色阴影的"廉价感"，紫调阴影更有"玉质" |
| 间距节奏 | **8px 基准网格**，主要节奏 8/16/24/32 | 标准移动端节奏 |
| 装饰元素 | **金色细线条 + 中式角花纹（仅四角点缀）+ 星曜符号** | 体现"古典智慧"调性，避免过度装饰 |

## 5. 风格参考案例（文字情绪板）

### 5.1 测测 App（紫微 / 星座板块）

- **整体感受**：神秘 / 年轻 / 卡片化 / 富有节奏
- **配色特点**：深紫 + 银白主调，金色作为会员标识
- **排版特点**：标题大字号，卡片内信息分层清晰，留白适度
- **组件风格**：圆角卡片，按钮带轻渐变，星座/星曜图标精致
- **动效特点**：滚动入场动画、星空粒子背景动效（轻微视差）
- **值得借鉴**：
  - 星盘 / 命盘的可视化层次（外圈宫位 + 内圈中心信息）
  - 卡片化解读分维度展示
  - 金色装饰线条点缀紫色卡片
- **需要规避**：
  - 过度依赖社区互动 UI（紫微灵犀 MVP 不做社区）
  - 部分页面信息密度过高

### 5.2 文墨天机（命盘可视化部分）

- **整体感受**：专业 / 工具感 / 视觉陈旧 / 信息密集
- **配色特点**：米白底 + 黑色文字 + 红色标记（重点星）
- **排版特点**：极致信息密度，宫位内多层文字叠放
- **组件风格**：传统正方形 12 宫格，无圆角
- **动效特点**：几乎无动效
- **值得借鉴**：
  - **传统正方形 12 宫格的信息组织方式**（4x4 去四角排列）
  - 宫位内"宫名 + 天干地支 + 主星 + 辅星"的分层逻辑
- **需要规避**：
  - 视觉陈旧（颜色单调、无层次）
  - 信息密度过高新手劝退
  - 全部用红色标记重要信息（与现代设计冲突）

### 5.3 多邻国 / Duolingo（积分 + 游戏化部分，作为积分系统参考）

- **整体感受**：友好 / 鼓励 / 有趣 / 节奏明快
- **配色特点**：明亮主色 + 大色块对比
- **排版特点**：大数字 + 简短文案，奖励反馈强烈
- **组件风格**：圆角按钮、3D 拟物图标（吉祥物）、签到日历
- **动效特点**：动画反馈丰富（连胜动画、积分跳动、勋章解锁）
- **值得借鉴**：
  - **签到打卡的视觉强化**（连续签到日历 + 周奖励）
  - **积分获得的动画反馈**（数字跳动 + 金币飞入）
  - **看广告 / 完成任务的明确奖励路径**
- **需要规避**：
  - 过于活泼的卡通风格（与东方雅致调性冲突）
  - 不要直接照搬卡通吉祥物

### 5.4 小红书（卡片视觉节奏，作为分享卡片参考）

- **整体感受**：清新 / 信息适度 / 视觉精致 / 易于分享
- **配色特点**：白底 + 高质量配图 + 红色品牌强调
- **排版特点**：图文穿插、留白克制、字号统一
- **组件风格**：圆角卡片，浅阴影
- **值得借鉴**：
  - **分享卡片的视觉节奏**（适合截图传播）
  - **标题字体大小与正文层级**
- **需要规避**：
  - 不照搬其品牌红色

### 5.5 苹果 / Apple（Hero 区视觉范式参考）

- **整体感受**：极简 / 高级 / 留白 / 文字驱动
- **配色特点**：单色背景 + 高对比标题
- **排版特点**：超大字号标题 + 精炼副文案
- **值得借鉴**：
  - **首屏 Hero 区的留白哲学**（标题独占一屏的气场）
  - 文字驱动而非图片堆砌
- **需要规避**：
  - 不要冷冰冰的科技感，要保留东方温度

## 6. 设计红线

### 6.1 必须避免

- **不要使用**廉价感的紫色（如 #800080 纯紫）— 必须用降饱和度的深紫
- **不要使用**纯白背景（#FFFFFF）— 必须用暖白 #FFF8F0 或暗紫，避免医院 / 办公室感
- **不要使用**大面积红色 — 命理语境会被读作"凶卦"
- **不要使用**卡通插画 / 萌系吉祥物 — 与"古典智慧"调性冲突
- **不要使用**过度饱和的金色（如 #FFD700）— 显廉价，必须用古金 #D4A84B
- **不要使用**多种字体混用（超过 2-3 个字体家族） — 破坏视觉一致性

### 6.2 色彩禁忌

- **绿配紫**（中国传统配色禁忌）
- **白底纯黑**（医院/办公感）
- **荧光色任意使用**（与雅致调性冲突）
- **多色 CTA**（一个页面只能有 1 个金色 CTA，否则失去焦点）

### 6.3 风格陷阱

- **过度玻璃态（Glassmorphism）**：会失去东方厚重感
- **过度新拟态（Neumorphism）**：与命盘可视化冲突
- **欧美科技感渐变**（如紫粉渐变）：会让产品显得像西方占卜
- **过度动画**：流畅但要克制，避免页面"跳动感"
- **直接照搬太极/八卦图**：紫微斗数体系并不主用八卦图，混用会显示专业性不足；可用十二地支符号、星曜符号替代

### 6.4 必须保留的"东方韵味"信号

- 暖色调背景（不是冷白）
- 金色细线条点缀（不是大面积）
- 中式标点符号习惯（如「」而非 ""）
- 命盘可视化保留传统正方形宫位结构（不要改成圆形星盘）
- 思源宋体标题营造书法感
- 图标可融入星曜符号、天干地支文字而非纯几何
