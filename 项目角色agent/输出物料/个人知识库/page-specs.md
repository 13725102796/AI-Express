### 页面状态列表

| 状态 | 描述 | 视觉表现 |
|------|------|---------|
| 默认-登录 | Tab 选中「登录」，显示邮箱+密码表单 | 登录 Tab 高亮 |
| 默认-注册 | Tab 选中「注册」，额外显示确认密码字段 | 注册 Tab 高亮 |
| 表单校验错误 | 邮箱格式错误/密码不一致/空字段 | 输入框红色边框 + 错误提示文字 |
| 登录中 | 提交表单后等待响应 | 按钮显示 loading spinner，表单不可编辑 |
| 登录失败 | 密码错误/账号不存在 | 表单顶部显示红色 toast 提示具体错误 |
| OAuth 加载中 | 点击第三方登录后 | 对应按钮 loading 状态 |

### 交互逻辑

| 编号 | 触发 | 过程 | 结果 |
|:----:|------|------|------|
| I-01-1 | 用户切换 Tab 到「注册」 | 表单平滑过渡，显示确认密码字段 | 按钮文案变为「创建账户」 |
| I-01-2 | 用户点击「登录」按钮 | 前端校验 > 发送请求 > 等待响应 | 成功：跳转 AI 问答主页；失败：显示错误提示 |
| I-01-3 | 用户点击 OAuth 按钮 | 新窗口/重定向到第三方授权页 | 授权成功回调后跳转主页；取消或失败显示提示 |
| I-01-4 | 用户输入邮箱后失焦 | 前端正则校验邮箱格式 | 格式错误时输入框变红，下方显示错误文案 |
| I-01-5 | 用户点击「忘记密码」 | 跳转密码重置页面（输入邮箱发送重置链接） | 显示"重置链接已发送到您的邮箱" |

### 页面元素清单

| 元素 | 类型 | 内容/数据 | 交互行为 | 状态变化 |
|------|------|----------|---------|---------|
| 知识空间选择器 | dropdown | 当前选中空间名 + 下拉列表 | 点击展开下拉选择空间 | 收起/展开；选中态 |
| 对话消息容器 | scrollable area | 消息列表 | 新消息自动滚动到底部 | - |
| 用户消息气泡 | message bubble | 用户输入的问题文本 | - | - |
| AI 回答卡片 | card | 回答正文 + 引用标注[1][2] | 流式输出打字机效果 | 生成中/生成完成 |
| 引用标注 pill | badge/pill | [1] [2] 等编号 | 悬浮显示来源预览浮层；点击跳转原文 | default/hover |
| 引用来源列表 | list | 文件图标 + 文件名 + 段落摘要 | 点击跳转知识条目详情 | - |
| 反馈按钮组 | button group | 有帮助(thumbs up) / 无帮助(thumbs down) | 点击提交反馈 | 未选/已选 |
| 无相关资料提示 | alert card | "知识库中未找到与该问题相关的资料" | 显示「上传相关资料」按钮 | - |
| 输入框 | textarea | placeholder: "基于你的知识库提问..." | Enter 发送，Shift+Enter 换行 | default/focus/disabled(生成中) |
| 发送按钮 | button[primary] | 发送图标 | 点击发送消息 | default/disabled(输入为空或生成中) |
| 新对话按钮 | button[ghost] | "+ 新对话" | 清空当前对话，开始新会话 | - |
| 对话历史列表 | list（侧边栏内） | 历史对话标题列表 | 点击切换到历史对话 | 选中态高亮 |

### 页面状态列表

| 状态 | 描述 | 视觉表现 |
|------|------|---------|
| 空状态（首次进入） | 无对话历史 | 居中显示欢迎语 + 示例问题引导卡片（3-4 个建议问题） |
| 等待输入 | 有历史对话，等待用户提问 | 对话列表正常展示，输入框获焦 |
| AI 生成中 | 已发送问题，AI 正在回答 | 消息区底部显示 AI 头像 + 打字机动画；输入框置灰不可输入；发送按钮变为停止按钮 |
| 生成完成 | AI 回答完毕 | 显示完整回答 + 引用来源 + 反馈按钮；输入框恢复可输入 |
| 生成超时 | 超过 30 秒未返回 | 显示加载状态 + "回答生成中，请稍候"；超 60 秒显示「重试」按钮 |
| 无相关资料 | 知识库无匹配内容 | 显示提示卡片 + 建议上传资料的引导 |
| 知识库为空 | 用户未上传任何资料 | 显示"请先上传资料到知识库"引导，附上传按钮 |

### 交互逻辑

| 编号 | 触发 | 过程 | 结果 |
|:----:|------|------|------|
| I-02-1 | 用户在输入框输入文本并按 Enter | 校验非空 > 追加用户消息气泡 > 输入框清空置灰 > 调用 AI 问答接口 | AI 回答以流式打字机效果逐字输出，完成后显示引用来源和反馈按钮 |
| I-02-2 | 用户按 Shift+Enter | 输入框内换行 | 不触发发送 |
| I-02-3 | 用户悬浮引用标注 [1] | 200ms 延迟后弹出浮层，显示来源文件名 + 段落摘要预览 | 鼠标移出后 300ms 浮层消失 |
| I-02-4 | 用户点击引用标注 [1] | 判断来源类型：PDF 打开预览定位到页码；网页跳转详情页 | 弹出知识条目详情侧抽屉或跳转到 P-04 |
| I-02-5 | 用户点击引用来源列表中的某条来源 | 跳转到该知识条目详情页 | 导航到 P-04，高亮引用段落 |
| I-02-6 | 用户点击「有帮助」/「无帮助」 | 发送反馈到后端 | 按钮高亮选中态，对侧按钮置灰 |
| I-02-7 | 用户切换知识空间选择器 | 下拉展开空间列表，点击选择 | 更新选中空间标识，后续提问在选定空间内检索 |
| I-02-8 | 用户点击「新对话」按钮 | 清空当前对话区域 | 显示空状态欢迎语 + 建议问题 |
